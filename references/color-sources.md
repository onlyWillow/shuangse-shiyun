# Chinese Color Source Notes

Use these sources when a task requires retrieval, citation, or verification of Chinese traditional color names:

- 中国色: https://zhongguose.com/
- Chinese Color Atlas: https://chinesecoloratlas.com/colors

Recommended practice:

1. Extract the core image color first.
2. Search the sources for nearby colors by hex/name when possible.
3. If exact lookup is unavailable, compare against a known palette and state that the selected name is the nearest match.
4. If sources disagree, prefer the source the user named. Otherwise prefer a color whose hue, saturation, and lightness match the extracted color over a poetic name that is visually farther away.
